# VitalHome Salud Design System (Standards)
# Name: vitalhome-design
# Description: Estándares visuales y de UX/UI para el proyecto VitalHome Salud para asegurar consistencia premium.

## Reglas de Tipografía

### Títulos (H2)
- **Color Principal:** Siempre usar el azul corporativo profundo `#0b3164`. Evitar el negro puro (`#000000`) o el gris muy oscuro (`text-secondary` por defecto) para mantener la identidad médica vibrante.
- **Fuente:** `Outfit` (font-heading).
- **Peso:** `font-bold` o `font-black`.
- **Efectos:** Se permite el uso de gradientes sutiles o resaltados en `text-primary` para palabras clave.

### Cuerpo de Texto
- **Color:** `#0b3164` con opacidad (70% o 80%) para legibilidad.
- **Fuente:** `Inter` (font-body).

## Reglas de Color
- **Primario:** `#104892` (Azul VitalHome).
- **Primario Oscuro:** `#0b3164` (Azul para títulos y elementos de autoridad).
- **Acento:** `#155ec0` (Azul vibrante para CTAs secundarios y efectos).
- **Éxito (Verde):** Usar verdes suaves para validación y resultados positivos de VitalHome.
- **Alerta (Rojo):** Usar rojos sutiles (`red-600/60`) para comparar con la competencia o señalar riesgos externos.

## Botones (btn-brand)
- Siempre deben incluir el efecto "Shine Hover" (gradiente que se desplaza y brillo a 45 grados).
- Curvatura: `rounded-full` para botones de acción principal.

## Animaciones
- Usar `animate-float` para elementos de confianza y autoridad para dar sensación de modernidad y dinamismo.
