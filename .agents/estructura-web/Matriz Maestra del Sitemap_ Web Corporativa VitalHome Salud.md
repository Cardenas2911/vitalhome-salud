# Documento Maestro: Web Corporativa VitalHome Salud (Versión 3.0)

Este documento consolida la arquitectura de información, el copywriting estratégico, las recomendaciones de CRO y el marcado Schema para la Web Corporativa de **VitalHome Salud**. Ha sido reestructurado para reflejar la nueva estrategia de **7 páginas de servicios únicos**, priorizando la autoridad médica y eliminando la fricción de precios para enfocar todo el ecosistema hacia la conversión directa mediante contacto.

## 3. Directrices de CRO y SEO (Sin Precios)

### Estrategia de "Precio Cero Visibilidad"
*   **Foco en el Valor:** Se prohíbe mostrar tablas de precios. En su lugar, se utilizan frases como: *"Planes personalizados según la necesidad del paciente"* o *"Consulte su presupuesto a medida"*.
*   **Embudo de Conversión:** Todo el tráfico debe dirigirse a un formulario de **"Valoración Inicial Gratuita"**. Esto permite captar el lead y realizar una venta consultiva telefónica.

### Optimización SEO y Entidades
*   **Entidades Clave:** `atención domiciliaria`, `enfermería profesional`, `geriatría`, `rehabilitación física`, `bienestar integral`, `adulto mayor`, `postoperatorio`, `cuidados pediátricos`, `médico domiciliario`, `telemedicina`, `monitoreo 24/7`, `apoyo psicológico`, `terapia ocupacional`, `masajes terapéuticos`, `entrenamiento adulto mayor`.
*   **Schema Markup:** Uso obligatorio de `Service` anidado en `Organization` para cada una de las 7 páginas, utilizando la propiedad `offers` con `priceCurrency: "COP"` y `description: "Consulta personalizada"`.
